  • Gameplay: Each player is given a hand of 3 cards in the game. The hands are then compared, and the person with the best hand wins. Players will add all 3 cards together, face cards are 10s, aces are 1s. The last digit of the sum is then the value of the hand. The suits do not play any role; all suites are the same. For example, if a player has Jack, 7, 5, which added up to be 22, then the value of the hand is 2. Value of the hand will be from 0 to 9. The player that has the higher score wins the game.
